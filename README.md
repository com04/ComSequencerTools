# ComSequencerTools

UE4.22以降で旧式の文字列でイベントを投げるEventTrackを使えるようにするプラグインです。  
  
4.21からシーケンサーのEventTrackがDirector形式になり出来ることが増えたものの、シーケンサーの数が多くなってくると逆に手間になってきます。  
また、4.20からバージョンアップした際の旧仕様との互換性を持たせたい等に。  
  
+ Event Track(String)を選択してください。
